# Design Notes

This scaffold uses a temporary Yozu visual direction so Vercel deployment can proceed before final logo review.

## Current Direction

- Citrus memory cue without making the product look like a food or juice app.
- Warm paper background, citrus accent, dark operator-grade panels.
- Messaging emphasizes travel decision and execution coordination, not OTA pricing or inventory control.

## Review Boundary

Final logo, favicon, deck visuals, QR codes, and screenshots must be reviewed by Loretta, QA, and Legal before external use.
